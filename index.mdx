## API Architecture

Our API follows RESTful principles and returns JSON responses. All requests must be made over HTTPS.

```json
{
  "success": true,
  "data": {
    "studios": [...],
    "pagination": {
      "page": 1,
      "limit": 20,
      "total": 150,
      "hasNext": true
    }
  },
  "meta": {
    "timestamp": "2025-01-15T10:00:00Z",
    "version": "1.0.0"
  }
}
```
